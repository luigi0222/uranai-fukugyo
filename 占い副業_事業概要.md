# 占い副業 - AI完全自動化モデル

## コンセプト
稼働を極限まで減らし、Claude APIの感情表現力をフル活用した  
**24時間365日・自動納品の高品質鑑定書ビジネス**

---

## 自動化フロー

```
【集客】SNSショート動画 / プラットフォーム掲載
    ↓
【受付】生年月日・性別・具体的な悩みを入力フォームで受取
    ↓
【決済】Stripe / プラットフォーム決済（初期は手動コピペも可）
    ↓
【生成】Claude API → 1万文字超の本格鑑定書を爆速生成
    ↓
【納品】自動メール or LINEでHTMLファイル(PDF変換可)を送付
```

## セットアップ手順

```bash
# 1. 依存ライブラリをインストール
pip install -r requirements.txt

# 2. Claude API キーを環境変数にセット
export ANTHROPIC_API_KEY="sk-ant-xxxxxxxxxxxxxxxxxx"

# 3. スクリプトを実行
python generate_fortune.py
```
